# ME_EM

## Tech Stack
- Frontend: React + Vite, Tailwind CSS, React Router DOM, Axios, Lucide React
- Backend: Node.js, Express.js, MongoDB (Mongoose), JWT, Multer

## Project Structure

- `client/`: frontend app
- `server/`: backend API

## Hướng dẫn chạy
npm install
### Server

```bash
cd server
npm run dev
```

### Client

```bash
cd client
npm run dev
```
